## Design Notes
- AI Prompt: You're a Solutions Architect building out a new mixed-media website for a New York Bestseller authoer who wants to bring her memoir to life with 2D and 3D images, integrated playlist of songs, and animated texts. Design a layout that can be easily written, edited, and published via Wordpress.

## Architecture
- Content Layer: Headless Wordpress
  1. Each chapter is written as a WP post with a category like "chapter". No additional uplift for the author using standard WP editor.
  2. Each chapter's hero photo will be a featured image.
  3. Each chpater will leverage WP's block editor's built-in "excerpt" to animate the pull-quote.
  4. A separate post category called "about" will power the About section (i.e. author's page).
  5. Website will update automatically (cached + revalidated on load) whenever the author hits "publish" in Wordpress.
- Delivery Layer: Integrate React site
  1. Supabase Edge Function will proxy the API calls so the site ID stays server-side and responses can be cached.
  2. Lovable WordPress.com connector will fetch each post via wp/v2/posts and handle auth + token refresh.
- Experience Layer: Cinematic and animated with slow Ken Burn transitions.
  1. Hero - full-bleed image, animated Serif titla, slow Ken Burns zoom, scroll-clue.
  2. Chapters - scrool-snapped sections, fading/sliding in via IntersectionObserver. 3D-tagged images get mouse-tilt parallax.
  3. Playlists - embedded sticky/floating Spotify (or YouTube) player
  4. About - includes portrait of author + bio + NYT bestseller mark
  5. Typography
     a) Display: Cormorant Garamond
     b) Body: Inter
     c) Palette: Deep navy/charcoal with warm gold accent, film-grain overlay
  7. Motion: Smooth scroll with reduced or slow motion
  8. Design System Tokens
     - '--ink'
     - '--parchment'
     - '--gold'
     - '--mist'
     - '--gradient-dusk'
     - '--gradient-spotlight'
     - '--shadow-cinematic'
     - '--transition-silk'
